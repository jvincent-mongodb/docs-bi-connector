Overview
--------

The {+odbc-driver+} provides connectivity between a SQL client
and the |bi|. The {+odbc-driver-name+} enables users to create a
:doc:`Data Source Name (DSN) </tutorial/create-system-dsn>` and connect
a variety of :doc:`BI tools </client-applications>` to the |bi-short|. It includes
the |bi-short| :doc:`authentication plugin </reference/auth-plugin-c>`,
which implements the client side of MongoDB-supported authentication
mechanisms.

.. note::

   The {+odbc-driver+} connects the DSN to the BI Connector.
   
   .. figure:: /images/bi-connector/components-all.jpg
      :figwidth: 700px
      :align: center
      :alt: Diagram showing that the DSN connects to the BI Connector.

Compatibility
-------------

The {+odbc-driver+} is compatible with |bi-short| version 2.5 and
later. The latest production version of |bi-short| is available from
the {+download-center+}.

-------------------

The {+odbc-driver+} is available for:

Workaround for Unsupported Platforms 
-------------------------------------------

- Ubuntu 22.04

  a. Download and install the `community client plugins
     <https://dev.mysql.com/get/Downloads/MySQL-8.4/mysql-community-client-plugins_8.4.0-1ubuntu22.04_amd64.deb>`__.
  
     .. code-block:: shell 

        sudo dpkg -i mysql-community-client-plugins_8.4.0-1ubuntu22.04_amd64.deb 
        sudo dpkg -i mysql-connector-odbc_8.4.0-1ubuntu22.04_amd64.deb 
        sudo dpkg -i mysql-connector-odbc-setup_8.4.0-1ubuntu22.04_amd64.deb

  #. Enable the Cleartext plugin. 

     .. code-block:: shell 

        export LIBMYSQL_ENABLE_CLEARTEXT_PLUGIN=1

  #. Download the `driver <https://dev.mysql.com/get/Downloads/Connector-ODBC/8.4/mysql-connector-odbc_8.4.0-1ubuntu22.04_amd64.deb>`__.
  #. Downlaod the `setup <https://dev.mysql.com/get/Downloads/Connector-ODBC/8.4/mysql-connector-odbc-setup_8.4.0-1ubuntu22.04_amd64.deb>`__.

- RHEL 8, 9

  .. note:: 

     For RHEL 8+, `download the ODBC Driver
     <https://dev.mysql.com/downloads/connector/odbc/>`__ published by
     Oracle MySQL with the version that supports RHEL 8.3 or later.

     For RHEL 9, `download the ODBC Driver
     <https://dev.mysql.com/downloads/connector/odbc/>`__ and install
     the ODBC Driver published by Oracle MySQL, set up RPMs, and enable
     the Cleartext plugin. 

Using the {+odbc-driver-name+}
--------------------------------------------------

Configure your system to use the {+odbc-driver+} by setting up a
:doc:`DSN </tutorial/create-system-dsn>`. A DSN holds configuration
information about your |bi-short| instance, including server address,
port number, and login credentials. See
:doc:`Create a System DSN</tutorial/create-system-dsn>` for
instructions on setting up a DSN on your system.
